Fiz Um mod para Minecraft

Fiz após anos sem tocar no assunto um mod para Minecraft Ultilizei nesse mod a api do forge. Então sobre o’que é esse mod.

A umas semanas atrás estava Jogando com uns colegas e estava com um projeto de remover a água de um oceano, porém o processo seria muito lento com as esponjas do minecraft vanilla, então decidi fazer um mod que adicionasse mais tipos de esponjas. O mod adiciona 5 novas esponjas, no total 10 blocos novos ao jogo pois é preciso ter uma versão seca e outra molhada do bloco, 2 dessas esponjas apenas expandem o alcance da esponja padrão do jogo e outras 3 são um tipo de esponja novo que absorve lava.

Foi necessário ler muitos exemplos de outros mod para entender como a api funcionava por isso para ter exemplos futuros eu criei um repositório com o código do mod para que eu possa vê lo no futuro.
